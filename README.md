# 🛒 Sahi E-commerce Platform

Full-stack e-commerce application developed using React.js, Spring Boot, and MySQL.

## Tech Stack

- React.js
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript

---

## SQL Implementation

This project uses MySQL for:

- User Management
- Product Management
- Cart Management
- Wishlist Management
- Order Management

Sample Queries:

```sql
SELECT * FROM users;
SELECT * FROM products;
SELECT * FROM orders;
```
