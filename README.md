
# 🛒 Sahi E-Commerce Platform

A full-stack e-commerce web application developed using React.js, Spring Boot, and MySQL.

This project demonstrates hands-on experience in:

✅ SQL Database Design  
✅ CSS-based Responsive UI Development  
✅ REST API Development  
✅ JWT Authentication  
✅ Cart, Wishlist, and Order Management  

## Tech Stack

- React.js
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript

---

## SQL Implementation

This project uses MySQL for:

- User Management
- Product Management
- Cart Management
- Wishlist Management
- Order Management

Sample Queries:

```sql
SELECT * FROM users;
SELECT * FROM products;
SELECT * FROM orders;
```
